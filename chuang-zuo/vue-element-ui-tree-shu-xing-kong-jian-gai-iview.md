把element-ui组件的tree树形控件迁移了一下，适配了iview组件

element-ui 的tree树形控件 修改为适配iview组件

## 使用方法

npm i chu-tree-iview

```
import chuView from 
'chu-tree-iview'


Vue.use(chuView)


<
chu-tree
>
<
/chu-tree
>
```

使用文档跟element-ui一模一样 http://element-cn.eleme.io/\#/zh-CN/component/tree

## 必须安装iview

样式风格全部替换成了ivew 功能全部按照element-ui原先一样
